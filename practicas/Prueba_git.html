codigos de git

(Todos estos comandos se ejecutan en la terminal)

  • Para crear un nuevo repostorio: git init

  • Para guardar cambios: git add (nombre del archivo)

  • Para hacer un commit: git commit (nombre del commit)

  • Para actualizar el github: git push origin master. Nota: si estas en alguna rama se pone el nombre de la rama.

  • Para crear una nueva rama: git branch (nombre de la rama)
